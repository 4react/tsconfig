# @4react / tsconfig
TypeScript configuration for React Modules

## Usage
#### Install dependency
```
npm i -D @4react/tsconfig
```

#### Add `tsconfig.json` to the project
```json
{
  "extends": "@4react/tsconfig"
}
```

#### Optionally override configuration
```json
{
  "extends": "@4react/tsconfig",
  "compilerOptions": {...}
}
```
